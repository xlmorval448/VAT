# Valorant Agent Tracker (VAT)

## Descripción de la aplicación
Esta es una WEB SPA diseñada para registrar y gestionar personajes (agentes) de Valorant. 

El propósito de la aplicación es ofrecer una interfaz donde los usuarios puedan añadir nuevos agentes especificando sus datos principales (nombre, rol, año de lanzamiento y una breve descripción), almacenando toda la información de forma persistente en la nube utilizando Firebase Firestore.

## Tecnologías de estilo
Para el diseño visual de la aplicación he utilizado **CSS** puro. utilizando Flexbox y CSS Grid para que el diseño sea responsive.
